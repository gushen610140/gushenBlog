# gushenBlog

## 项目介绍

个人博客站，用于展示个人所参与的项目，同时将分享技术文章。

## 技术栈
前端: Vue3 + Typescript + Vite

后端: Java + SpringBoot

数据库: MySQL

## 项目目录

```
├── src 主要文件
├── public 静态资源
├── README.md
```

## 项目运行

### 后端

请移步到 [gushenBlog-backend](https://gitee.com/gushenfree/gushen-blog-back-end)

### 前端

```
cd gushenBlogFrontEnd
npm install
npm run dev
```

## 项目部署

均部署在阿里云服务器
