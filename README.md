# gushenBlog

## 项目介绍

个人博客站，用于展示个人所参与的项目，同时将分享技术文章。

## 技术栈
前端: Vue3 + TypeScript + Vuex + Element

后端: SpringBoot + MybatisPlus + MySQL

## 项目目录

```
├── src 主要文件
├── public 静态资源
├── README.md
```

## 项目运行

### 后端

请移步到 [gushenBlog-backend](https://gitee.com/gushenfree/gushen-blog-back-end)

### 前端

```npm install```

```npm run dev```

## 项目部署

阿里云ECS
